# 🌱 Calculadora de Produção por Hectare

Aplicação desenvolvida para calcular a produtividade agrícola por hectare, auxiliando produtores, técnicos e estudantes na estimativa da produção de culturas agrícolas.

O sistema permite realizar cálculos de forma rápida e prática, fornecendo indicadores que auxiliam no planejamento e na tomada de decisões.

---

## 🎯 Objetivo

Facilitar o cálculo da produtividade agrícola através de uma ferramenta simples e intuitiva, reduzindo erros de cálculo manual e agilizando análises de produção.

---

## ✨ Funcionalidades

* Cálculo de produção por hectare
* Estimativa de produtividade agrícola
* Conversão automática de unidades
* Exibição dos resultados em tempo real
* Interface simples e intuitiva
* Validação dos dados informados

---

## 🚜 Aplicações

O sistema pode ser utilizado para:

* Planejamento agrícola
* Estimativa de safra
* Controle de produtividade
* Estudos acadêmicos
* Projetos agrícolas
* Análises técnicas de produção

---

## 🧮 Como Funciona

O usuário informa os dados da produção, como:

* Área cultivada
* Quantidade produzida
* Unidade de medida utilizada

O sistema realiza automaticamente os cálculos necessários e apresenta:

* Produção total
* Produção por hectare
* Indicadores de produtividade

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/AdautoFurich/calcular-producao-por-hectare.git
```

Entre na pasta:

```bash
cd calcular-producao-por-hectare
```

Abra o arquivo:

```text
index.html
```

Ou execute utilizando o Live Server do VS Code.
